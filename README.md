# GestureOS

Gesture-based interaction layer — control your system through hand and body gestures.

## Status

Early scaffolding. Architecture and feature set TBD.

## Getting started

```bash
git clone https://github.com/Hipdarius/GestureOS.git
cd GestureOS
cp .env.example .env   # fill in your keys
```
